About This Blog

Welcome to my blog! I'm John Vandivier, and this website is part of my "Trial by Fire v3" project, based on the open-source curriculum provided by Ladderly.io.

This project demonstrates my ability to create a responsive, interactive web application using fundamental web technologies: HTML, CSS, and JavaScript. It's a practical implementation of core web development concepts, including:

  • Semantic HTML structure
  • Modern CSS styling and layouts
  • Client-side JavaScript for dynamic content
  • Form handling and DOM manipulation
  • Responsive design principles

Feel free to explore the blog, create new posts, and connect with me on LinkedIn!
